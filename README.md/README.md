# Fitness Tracker - V1 Prototype

A full-stack web application prototype designed to track workouts, calculate progressive overload, and foster community through a "Squad Hub". 

## 🚀 Core Features
* **Dynamic Dashboard:** Real-time calculation of weight loss goals and required weekly rates.
* **Global Workout Overlay:** A persistent, Spotify-style background player that allows users to log sets without being locked into a single screen.
* **Interactive Vault:** A searchable database of exercises that can be dynamically injected into an active workout session.
* **History & Analytics:** Volume tracking and calendar-based heatmap generation for completed workouts.
* **Squad Hub:** A social feed for sharing post-workout updates and photos.

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (DOM Manipulation, Chart.js)
* **Backend:** Python
* **Database:** Firebase Firestore / LocalStorage (Hybrid state management)

## 📌 Project Status
**Archived / On Hold.** This repository serves as a V1 architectural prototype. The focus of this build was exploring complex UI state management (overlays, mini-players) and establishing a baseline pipeline between the frontend DOM and backend database.
